# HorizonOS

An immutable Linux distribution with intelligent automation and adaptive desktop experiences.

## Quick Start

```bash
# Build OSTree commit
sudo ./scripts/scripts/build-test.sh

# Build ISO
sudo ./scripts/scripts/build-iso.sh

# Test in QEMU
./test-iso.sh
```
